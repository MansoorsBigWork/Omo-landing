# OMO Landing

OMO Landing is the public landing page for **One Million Opportunities (OMO)**, a startup building a better graduate talent pipeline for students and employers. The site is designed to communicate OMO’s core mission, explain the product vision, and act as an entry point for students, employers, and early supporters.

## About OMO

One Million Opportunities (OMO) is focused on solving a major employability problem: students struggle to get meaningful work experience, while companies struggle to identify candidates who can actually do the job beyond what looks good on paper.

OMO’s product vision includes:
- **OMOships** — short, real-world work simulations built from actual company challenges
- **Jimmy** — an AI career co-pilot that helps students understand what is standing between them and the roles they want

This landing page is the first public-facing layer of that vision.

## Purpose of this repository

This repository contains the frontend code for the OMO landing page.

Its goals are to:
- Communicate OMO’s value proposition clearly
- Present the brand and messaging
- Support startup outreach and validation
- Give students and companies a simple way to understand the product
- Serve as the public web presence for the project

## Current status

This is an active early-stage startup landing page. It is being used as part of OMO’s public presence and broader founder materials, and will likely continue to evolve alongside the product, messaging, and customer discovery.

## Why this project matters

For an early-stage startup, the landing page is not just a marketing site — it is part of the validation and communication layer. It helps explain the problem, sharpen messaging, support outreach, and create credibility with students, employers, partners, and accelerator programs. We worked very hard to ensure the design language and overall layout reflected our vision.

## Getting started

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
